= Verificando el pipeline
include::_attributes.adoc[]

- La pestaña CI te permite seguir el progreso de la build pipeline (proceso automatizado que compila, prueba y despliega aplicaciones desde el código fuente hasta producción) de la aplicación.

image::CI_8.png[]

- Observa cómo las medidas de seguridad están integradas en el proceso de compilación, de acuerdo con las políticas establecidas por el equipo de seguridad. Estas medidas incluyen:

1. Análisis estático de código usando Sonarqube.

2. Verifica que el último commit, el cual activó la pipeline, está digitalmente firmado por una fuente de confianza.

3. Generación de SBOM (Software Bill of Materials), para obtener información sobre la composición de la aplicación.

4. Integración con ACS (Advanced Cluster Security) para escanear la imagen en busca de vulnerabilidades.

- Como desarrollador, no tienes que preocuparte de estos aspectos complejos.

- Todos ellos están preparados mediante el Golden Path, lo que garantiza un proceso de desarrollo ágil y seguro.

- De hecho, incluso podríamos sustituir Tekton por otra herramienta de CI, ¡y no tendrías que preocuparte por los detalles!

- Una vez completada la ejecución de la build pipeline, revisamos el estado de nuestra aplicación.
