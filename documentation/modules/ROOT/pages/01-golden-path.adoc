= Lanzando tu primera Golden Path Template (GPT)
include::_attributes.adoc[]

- Como nuevo desarrollador, se te ha pedido crear un proyecto para implementar un servicio basado en Quarkus.

- Para hacerlo, vas a usar una Golden Path Template (GPT) preconfigurada, que agilizará tu proceso de incorporación al nuevo proyecto.

1. La GPT creará dos repositorios en GitLab, uno para el código fuente de la aplicación y otro para los Manifiestos de Kubernetes.

2. El siguiente paso en la GPT instruirá a ArgoCD para crear una aplicación y todos los recursos especificados en los Manifiestos en el clúster de OpenShift de destino.

3. En este ejemplo, estamos utilizando Helm charts como entrada para desplegar con ArgoCD. Y un pipeline de CI en Tekton para compilar tu aplicación.

image::GPT_Flow.png[]

- Para lanzar la GPT, haz clic en *Crear* en el menú lateral.

- Se te mostrarán todas las GPT que el equipo de Ingeniería de la Plataforma ha configurado para ti.

- Haz clic en el enlace *Choose* en la parte inferior derecha del recuadro "Securing a Quarkus Service Software Supply Chain" para seleccionar esta plantilla.

image::Choose_GPT.png[]

- A continuación, se te pedirá que completes un formulario simple con información básica.

- Modifica el *Name* con tu nombre y apellidos *en minuscula*

> [!WARNING]  
> Tu nombre y apellidos tiene que ser todo en minuscula

- Mantén el resto de valores por defecto y haz clic en *Next* para continuar.

image::GPT_form_1[]

- En el segundo formulario, elije Quay para el Image Registry

image::GPT_form_2[]

- Mantén el resto de valores por defecto y haz clic en *Next* para continuar.

- En el tercer y último formulario, verás valores predeterminados para el servidor Git que aloja el repositorio del código fuente de tu aplicación.

image::GPT_form_3[]

- Mantén los valores por defecto y haz clic en *Review* para continuar.

- Se te presentará un resumen. Ahora puedes revisar que todo esté en orden y luego hacer clic en el botón *Create*

image::GPT_form_4[]

- Después de hacer clic en el botón Create, Red Hat Developer Hub utilizará la información que has proporcionado para iniciar el proceso de configuración según lo descrito en la GPT seleccionada.

- También registrará tu aplicación en el Catálogo.

- Ahora, continúa haciendo clic en el enlace *Open Component in catalog*.
