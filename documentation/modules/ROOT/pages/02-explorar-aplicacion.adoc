= Explorando la nueva aplicación creada
include::_attributes.adoc[]

Este es el dashboard que te permite monitorizar el estado de la aplicación así como enlaces a las herramientas de desarrollo.

En este primer paso del laboratorio debemos esperar a que los componentes de construcción de la aplicación se desplieguen. Para ello, en la sección **Deployment summary** debemos esperar a que *<tu-nombre>-dev-build* esté *Synced* y *Healthy*.

image::Explorar-app-2.png[]


Una vez que *<tu-nombre>-dev-build* esté *Healthy*, puedes continuar al siguiente paso *Continuous Integration (CI)*.
